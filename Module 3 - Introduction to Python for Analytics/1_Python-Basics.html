6. Classes and Objects¶

Python is an Object-Oriented Programming (OOP) language. Almost everything is an Object, and each object belongs to a Class. More on this here - https://docs.python.org/3/tutorial/classes.html

# objects can also have "Methods" - think of these as actions on the object

class Dog:
    def __init__(self, fur_color='golden', size_of_dog='large', status='sitting'):
        self.fur = fur_color
        self.size = size_of_dog
        self.status = status
    
    def run(self):
        self.status= 'running'
        
    def sit(self):
        self.status = 'sitting'
        
    def bark(self, n_barks):
        return n_barks*"Bark"
